# Scripts de migracion a Confluence

Esta carpeta queda reservada para los scripts del proceso de sincronizacion hacia Confluence Cloud.

Contenido esperado en proximas iteraciones:

- script CLI principal de publicacion
- utilidades de conversion Markdown -> formato `storage`
- validadores de links internos
- manejo de mapeo local `archivo -> pageId`

Mantener en esta carpeta solamente scripts y helpers ligados a la migracion/publicacion, para no mezclar esta automatizacion con el proceso de relevamiento documental.
